# The Log-Linear Failure Rate (Log-LFR) Distribution- R code Repository
This repository contains the R codes used in the research work entitled " The Log-Linear Failure Rate Distribution: Properties, Estimation and Application to Aircraft
Windshield Failure and Service Data".
The repository includes:
# Graphical analysis of various distributions:
Code for plotting the Probability Density Function (PDF), Cumulative  Distribution Function (CDF) and Hazard Rate Function (HRF).
# Stochastic ordering analysis: 
Including likelihood ratio ordering and usual stochastic ordering for a series system of the Log-LFR distribution, with illustrative examples and counterexamples.
# Simulation:
Code for parameter estimation of the Log-LFR distribution using Monte Carlo simulation.
# Application to real lifetime datasets:
Code for two real datasets, including TTT plot, P–P plot, Q–Q plot, and fitted histograms to assess the goodness-of-fit of the Log-LFR distribution.

These codes allow readers to reproduce the simulation study, graphical illustrations, stochastic ordering results, and
real data analysis presented in the associated manuscript.
